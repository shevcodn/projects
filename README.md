## Project-01: Stock Portfolio Tracker

**Python CLI app** — track your stock portfolio in the terminal.

```
Stack:   Python · Alpha Vantage API · LinkedList · HashMap
Status:  🔄 In Progress
```

**Features:**
- `add TICKER QTY PRICE` — buy shares, auto-calculates avg price
- `sell TICKER QTY` — sell with validation
- `portfolio` — full table with real-time P&L
- `history TICKER` — transaction history via LinkedList
- `top N` — top N positions by profit

**Data structures used:** HashMap for holdings, LinkedList for transaction history

**API:** Alpha Vantage — real market prices, cached every 65 min
